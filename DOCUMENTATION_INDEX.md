## 🎯 Quick Reference

### Commands

```bash
# Development
npm install              # Install dependencies
npm run dev              # Start dev server
npm run lint             # Check code quality
npm run lint --fix       # Fix code issues

# Production
npm run build            # Build for production
npm run preview          # Preview production build

# Deployment
vercel                   # Deploy to Vercel
netlify deploy --prod    # Deploy to Netlify
```

### File Locations

```
src/components/          # React components
src/assets/              # Images and static files
src/lib/                 # Utility functions
public/                  # Static files (robots.txt, sitemap.xml)
dist/                    # Production build output
```

### Key Files to Edit

- Company name: `src/components/Header.jsx`, `src/components/Footer.jsx`
- Contact email: `src/components/Cta.jsx`, `src/components/Footer.jsx`
- Services: `src/components/WhatWeDo.jsx`
- Benefits: `src/components/WhyChooseUs.jsx`
- Colors: `src/index.css`
- Images: `src/assets/`
