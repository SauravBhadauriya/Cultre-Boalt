## 🔧 Customization

### Update PR Data

Edit `src/data/prData.js`:

```javascript
export const prData = {
  overview: {
    title: "PR",
    subtitle: "Your Subtitle",
    description: "Your description"
  },
  services: [
    {
      id: 1,
      title: "Service Name",
      description: "Service description",
      icon: "📰"
    }
  ],
  campaigns: [
    {
      id: 1,
      title: "Campaign Title",
      category: "Category",
      image: "/path/to/image.jpg",
      description: "Campaign description",
      coverage: "15+ Publications",
      reach: "2.5M+ Impressions",
      year: 2024
    }
  ]
  // ... more data
}
```

### Add New Campaign

1. Add to `campaigns` array in `src/data/prData.js`
2. Include all required fields
3. Update portfolio items if needed

### Add New Influencer

1. Add to `influencers` array
2. Include name, followers, category, image

### Change Colors

Edit `src/pages/PR.jsx`:
- Hero background: `from-slate-800 to-slate-900`
- Accent color: `bg-blue-600`
- Modal header: `from-blue-600 to-blue-700`
